# 视频通话

## 场景概述

基础多人视频通话，通过输入房间号即可快速体验高清视频通话。支持自定义视频分辨率、帧率、音质、美颜、切换布局等功能

## 前提条件

- 已在控制台创建应用，并获取了应用对应的 App Key。
- 已成功开通 音视频通话 2.0 产品。

详细操作请参考[开通服务](https://doc.yunxin.163.com/docs/jcyOTA0ODM/DkyMDM2Mzk?platformId=50002)。
